This software requires a config file to works !

# Setup configuration file
Create a file ``config.yml`` in the same directory than .exe

```yml
ftp:
  server: ""
  mods-directory: "/games/FarmingSimulator22/mods/"
  username: ""
  password: ""
server:
  ip: "http://XX.XX.XX.XX:XXXX"
  code: ""
mods-directory: ""
```
